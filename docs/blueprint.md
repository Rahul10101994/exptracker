# **App Name**: FinTrack

## Core Features:

- Balance Display: Show current balance with income, expense, and savings rate.
- Investment Overview: Provide a summary of investments, including active investments and average progress.
- Spending Trend: Display spending trend over the last 7 days in a chart format.
- Transaction Input: Allow users to add transactions using a plus button, logging all expenses.

## Style Guidelines:

- Primary color: Soft, desaturated light blue (#B0E2FF) for a calm, trustworthy feel.
- Background color: Light gray (#F5F5F5) to provide a clean, modern backdrop.
- Accent color: A slightly more saturated green-blue (#7ECEFD) to highlight interactive elements.
- Body and headline font: 'Inter' sans-serif font for clear readability and a modern, neutral feel.
- Use simple, geometric icons in a light color to represent financial data.
- Card-based layout with rounded corners to organize information and create visual hierarchy.
- Smooth transitions and subtle animations on data updates.
